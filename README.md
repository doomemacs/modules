This repository houses [Doom Emacs](https://doomemacs.org)' official starter
kit: a library of official modules that can serve as a foundation for your Emacs
configuration by:

- Pre-configuring support for a variety of programming languages, tools, and
  operating systems,
- Fixing common issues with Emacs packages or their interopability with other
  packages.
- Optimizing Emacs and third party packages,
- Providing sensible defaults for packages and features to spare you the months,
  maybe years, you'd otherwise spend discovering yourself.

Modules here are carefully curated by Doom's author and core maintainers.

> [!IMPORTANT]
> Each module possesses their own documentation, accessible within Doom via `M-x
> doom/help-modules` (or `M-x +lookup/documentation` while your cursor is on a
> module's name in `$DOOMDIR/init.el`).


## Install

Until Doom core v3 is released, this repo exists as a git submodule of
doomemacs/core and doesn't need to be directly installed (yet).
